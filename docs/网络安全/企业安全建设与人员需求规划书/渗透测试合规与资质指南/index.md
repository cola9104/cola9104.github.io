# 渗透测试合规与资质指南

## **一、企业层面资质**

### **1. CNNVD 技术支撑单位**

- **定位**：国家信息安全漏洞库（CNNVD）认定的最高级别技术支撑单位。
- **价值**：代表漏洞研究、预警响应、协同研判的国家级能力；在政府、央企、大型甲方项目中是强有力的背书。
- **有效期**：1 年，年度考核续期或升降级。
- **一级技术支撑单位**
- **定位**：国家级最高等级，承担漏洞研究、预警响应、协同研判的核心任务。
- **要求**：
- **适用场景**：政府、央企、大型甲方项目。
- **有效期**：1 年，年度考核决定续期或升降级。
- **定位**：具备较强漏洞收集与分析能力，能承担行业级支撑任务。
- **要求**：
- **适用场景**：省级、行业级项目。
- **有效期**：1 年，年度考核续期或升降级。
- **定位**：基础支撑单位，主要参与漏洞报送与部分分析。
- **要求**：
- **适用场景**：中小企业、科研院所合作项目。
- **有效期**：1 年，年度考核续期或升降级。
### **2. 国家信息安全测评中心（CNITSEC）**

- **定位**：由 CNITSEC 颁发的信息安全服务资质，覆盖安全集成、运维、应急响应、风险评估、渗透测试等多个方向。
- **价值**：法律与行业规范下的合规资质，广泛用于政府与金融行业。
- **要求简述**：境内注册法人；固定场所与资金；配备一定比例的 **CISP/CISAW**持证人员；具备相关项目业绩。
- **一级资质**
- **定位**：最高等级，覆盖安全集成、运维、应急响应、风险评估、渗透测试等全方向。
- **要求**：
- **适用场景**：政府、金融、能源等高合规行业。
- **有效期**：3 年，需年度确认，期满换证。
- **定位**：具备较强安全服务能力，适合行业客户。
- **要求**：
- **适用场景**：一般行业客户。
- **有效期**：3 年，需年度确认，期满换证。
- **定位**：基础安全服务能力，适合中小企业。
- **要求**：
- **适用场景**：中小企业内部安全测试。
- **有效期**：3 年，需年度确认，期满换证。
CNITSEC 资质对 **CISP/CISAW 持证人员比例** 有明确要求。

- **一级**：需配备较高比例的 **CISP/CISAW**持证人员（一般要求 ≥30% 技术人员持证），并具备丰富项目业绩。
- **二级**：持证比例适中（约 15–20% 技术人员持证），项目业绩要求较少。
- **三级**：只需少量持证人员（约 5–10% 技术人员持证），满足基本安全管理制度即可。
### **3. CCRC 信息安全服务资质认证**

- **定位**：由 **中国网络安全审查技术与认证中心（CCRC）**颁发，评估企业在安全服务方面的综合能力。
- **价值**：在政府、金融、能源、医疗等关键行业的安全服务与招投标中广泛使用；风险评估、应急处理方向与渗透测试直接相关。
- **要求简述**：
- 企业需具备固定场所、资金与安全管理制度。
- 配备专职技术人员，通常要求一定比例的 **CISP**持证人员。
- 有相关安全服务项目业绩与交付能力。
- **定位**：最高等级，综合安全服务能力，广泛用于政府、金融、能源、医疗等关键行业。
- **要求**：
- **适用场景**：高合规场景、招投标必备。
- **有效期**：3 年，需年度监督与维持。
- **定位**：具备较强安全服务能力，适合行业客户。
- **要求**：
- **适用场景**：行业客户、区域性项目。
- **有效期**：3 年，需年度监督与维持。
- **定位**：基础安全服务能力，适合中小企业。
- **要求**：
- **适用场景**：中小企业、基础合规。
- **有效期**：3 年，需年度监督与维持。
CCRC 资质同样要求企业配备一定比例的 **CISP 持证人员**。

- **一级**：持证比例最高（通常 ≥30% 技术人员持证），且需有丰富的安全服务项目业绩。
- **二级**：持证比例适中（约 15–20% 技术人员持证），项目业绩要求较少。
- **三级**：只需少量持证人员（约 5–10% 技术人员持证），满足基础安全管理制度即可。
## **二、个人层面证书（公司无资质时的替代方案）**

- **CISP 系列**（注册信息安全专业人员）
- 包括 CISP、CISP-PTE（渗透测试方向）、CISM、CISE、CISD 等，国内认可度最高。
- 信息安全工程师、网络工程师等，国家职业资格考试，证明专业能力。
- CEH（Certified Ethical Hacker）、OSCP（Offensive Security Certified Professional），在国际渗透测试领域认可度高。
- PMP（项目管理专业人士认证），在大型安全项目中证明管理与交付能力。
## **三、总结与应用场景**

- **公司持有 CNNVD / CNITSEC / CCRC 一级资质**→ 可在政府、金融、能源等高合规场景提供渗透测试服务，具备国家级认可。
- **公司暂未持有资质，但个人持有 CISP-PTE / CEH / OSCP 等证书**→ 可在商业项目、企业内部安全测试中提供服务，满足基本合规要求。
- **招投标建议**：在资质条款中明确写明“具备 CNNVD 一级技术支撑单位或 CNITSEC/CCRC 信息安全服务一级资质”，并附证书编号与有效期；若无企业资质，可强调团队成员的 CISP/CEH/OSCP 等证书。

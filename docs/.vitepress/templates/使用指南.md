# CategoryPage 组件使用指南

## 📖 简介

`CategoryPage` 是一个可复用的页面模板组件，用于统一所有主分类页面的样式。所有分类页面（如网络安全、嵌入式安全、渗透测试等）都应使用此组件。

## ✨ 特性

- 🎨 统一的页面样式和布局
- 💡 突出显示的简介区域
- 🎯 自动美化标题和内容
- 📱 完全响应式设计
- 🔧 易于使用和维护

## 🚀 使用方法

### 1. 基本用法

```markdown
---
layout: doc
title: 页面标题
outline: deep
---

<CategoryPage intro="这里是简短的介绍文字，一到两句话概括本专栏的内容...">

# **🔍 第一个主题**

内容...

# **📊 第二个主题**

内容...

</CategoryPage>
```

## 📋 模板结构

### 标准结构

```markdown
---
layout: doc
title: 分类名称
outline: deep
---

<CategoryPage intro="简介文字...">

# **🔍 什么是XXX？**
定义和基本概念...

# **📊 XXX的重要性**
### **1. 子主题1**
- **要点1**：说明
- **要点2**：说明

### **2. 子主题2**
- **要点1**：说明
- **要点2**：说明

# **⚠️ XXX威胁/挑战**
威胁类型...

# **🛡️ XXX防护/解决方案**
防护措施...

# **🚀 XXX发展趋势**
发展方向...

# **💼 XXX职业发展**（可选）
职业路径...

# **📝 总结**
总结性内容...

**学习建议**
1. 建议1
2. 建议2
3. 建议3

</CategoryPage>
```

## 🎨 样式说明

### 简介区域样式

- 浅色渐变背景（蓝色到紫色）
- 左侧品牌色边框
- 圆角设计
- 较大的字号和行距

### 标题样式

- 一级标题带有底部边框
- 二级标题更小的间距
- 所有标题支持表情符号
- 自动对齐和间距

## 📝 注意事项

1. **必须包含闭合标签**：`</CategoryPage>` 必须在文件末尾
2. **intro 属性必填**：提供简短的介绍文字
3. **标题格式**：使用 `# **🔍 标题**` 格式（加粗+表情）
4. **列表格式**：关键词使用加粗 `**关键词**：说明`
5. **保持一致性**：所有主分类页面应保持相似的结构

## 🆘 常见问题

### Q: 组件不显示？
A: 确保已正确导入和注册组件，检查 `theme/index.js` 文件。

### Q: 样式不生效？
A: 清除缓存并重启开发服务器：`rm -rf .vitepress/cache && npm run docs:dev`

### Q: 如何修改样式？
A: 编辑 `CategoryPage.vue` 文件中的 `<style>` 部分。

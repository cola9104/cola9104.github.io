## 🏆 CTF题目类型

### Web安全
- **SQL注入** - 数据库注入攻击
- **XSS跨站脚本** - 客户端脚本注入
- **文件上传** - 恶意文件上传
- **代码审计** - 源代码安全分析

### 二进制利用 (Pwn)
- **栈溢出** - 栈缓冲区溢出
- **堆溢出** - 堆缓冲区溢出
- **格式化字符串** - 格式化字符串漏洞
- **ROP链** - 返回导向编程

### 密码学 (Crypto)
- **对称加密** - AES、DES等算法
- **非对称加密** - RSA、ECC等算法
- **哈希函数** - MD5、SHA等算法
- **数字签名** - RSA签名、ECDSA等

### 逆向工程 (Reverse)
- **静态分析** - 反汇编、反编译
- **动态分析** - 调试、跟踪
- **加壳脱壳** - 软件保护技术
- **反混淆** - 代码混淆对抗

### 杂项 (Misc)
- **隐写术** - 信息隐藏技术
- **取证分析** - 数字取证技术
- **编程算法** - 算法编程题目
- **网络分析** - 网络流量分析

## 📚 解题技巧

- **仔细阅读题目** - 理解题目要求

- **记录过程** - 记录解题步骤和思路
